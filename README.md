# Suspiria Website

Сайт-визитка рок-группы Suspiria.
Адаптивная вёрстка под мобильные устройства.
Размещён через GitHub Pages.

## 🚀 Функционал

* 🎵 Проигрывание музыки
* 📝 Гостевая книга / форма обратной связи
* 🖼️ Информация о группе, медиа и ссылки
* 🖼️ Адаптивная вёрстка под mobile

## 🛠️ Технологии

* HTML5
* CSS3
* JavaScript (vanilla)
* Formspree (для обработки формы)

## 📩 Обратная связь

Форма отправляет сообщения через Formspree без перезагрузки страницы (fetch API).

## 📦 Структура проекта

```
/index.html
/assets/
  /images/
  /gif/
  /audio/
/css/
/js/
  script.js
  form.js
  sidebar.js
```

## 🌐 Деплой

Сайт хостится через GitHub Pages.
